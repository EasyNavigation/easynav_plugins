## Plugin (pluginlib)

- **Plugin Name:** `easynav_routes_maps_manager/RoutesMapsManager`
- **Type:** `easynav::RoutesMapsManager`
- **Base Class:** `easynav::MapsManagerBase`
- **Library:** `easynav_routes_maps_manager`
- **Description:** Routes (no-op) Maps Manager that demonstrates the Maps Manager API. It forwards/republishes a basic occupancy map flow and exposes standard topics and a save-map service.
